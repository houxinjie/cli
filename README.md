# cli

A scaffold for projects.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x, 5.x preferred) and [Git](https://git-scm.com/).

``` bash
$ npm install -g https://github.com/houxinjie/cli
```

### Usage

``` bash
$ cli new <project-name>
```

Example:

``` bash
$ cli new my-project
```
